# ECOMMERCE

## This project concerns the creation of a rest end point and perform different tests over it

### Technologies used

* Spring Boot
* H2 Database
* Java
* Lombok
* Maven
* JUnit
* Hibernate
* JPA

### How to test

You can use Postman and do a GET call to "http://localhost:8082/price" URL, with the following params

* appDate: Timestamp
* productId: Integer
* brandId: Integer
